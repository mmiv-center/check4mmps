Assessment

Test if StudyDate is present. The "StudyDate" DICOM tag (0008,0020) should be present and in the correct format, e.g. 20060306 (sixths of march 2006).

Test if Study, Series and SOPInstanceUID tags are in each DICOM file. The three identifiers do not contain any patient relevant information. They are needed to distinguish a DICOM file from any other file (in the univers).

CSAImageHeaderInfo and CSASeriesHeaderInfo If SIEMENS is the manufacturer we expect the CSA header entries (private tags) to have a value.
